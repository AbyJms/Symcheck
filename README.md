# Symcheck 🩺

**Symcheck** is an AI-powered health bot designed to provide immediate symptom analysis and medical guidance. By leveraging the high-speed inference of the **Groq SDK** and the **Llama 3** model, it offers users likely causes for their symptoms along with suggested remedies.

## 🚀 Live Demo
Check out the live application here: [symcheck](https://symcheck-five.vercel.app/)

---

## ✨ Features
*   **AI Symptom Analysis**: Real-time chat interface to describe health issues and receive AI-generated insights.
*   **Common Symptom Quick-Cards**: Instant advice for common ailments like Headaches, Fever, Cough, and Fatigue.
*   **Remedy Suggestions**: Provides immediate home remedies and suggested over-the-counter medicines.
*   **Responsive UI**: A clean, modern interface designed for both desktop and mobile accessibility.

---

## 🛠️ Tech Stack
*   **Frontend**: HTML5, CSS3, JavaScript.
*   **Backend**: Node.js, Express.
*   **AI Engine**: Groq SDK (Llama 3 Model).
*   **Database**: PostgreSQL.
*   **Deployment**: Vercel.

*Symcheck is an AI-powered tool for informational purposes only. It does not provide professional medical advice, diagnosis, or treatment. Always seek the advice of your physician or other qualified health provider with any questions you may have regarding a medical condition.*
